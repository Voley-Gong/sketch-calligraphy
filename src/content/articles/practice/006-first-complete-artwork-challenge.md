## 为什么你需要这个挑战

你已经画了几十张速写练习——线条、形体、光影、动态、白描、水墨，各种技法都练过。但这些练习稿有一个共同特点：**它们是片段，不是作品。**

练习稿和作品之间隔着一道坎：

| 维度 | 练习稿 | 完整作品 |
|------|--------|----------|
| 时间 | 5-30分钟 | 2-10小时（可分多次） |
| 目标 | 练某个技法 | 传达一个完整的感受 |
| 构图 | 单个人物/局部 | 有主次、有层次、有留白 |
| 完成度 | 画到哪算哪 | 每个区域都经过推敲 |
| 结尾 | 放下笔就结束 | 签名、日期、可装框 |

很多学画的人卡在这个阶段：技法会了，但从来没逼自己走完一遍**完整创作流程**。这篇训练就是带你走第一遍。

> **核心理念：你的第一幅作品不需要伟大，但需要完整。** 完成比完美重要。

## 创作前的准备

### 工具清单

- **主笔：** 你最顺手的笔（铅笔、钢笔、中性笔、毛笔均可——选你控制力最强的）
- **备用笔：** 一支细笔（用于细节刻画）+ 一支粗笔（用于深色区域）
- **纸：** 16K或8K速写本（不要用A4打印纸，太薄且没有质感）
- **参考：** 1-2张你喜欢的照片或大师作品（作为灵感起点，不是复制对象）
- **心态：** 接受"第一幅可能不够好"——这完全正常

### 选定你的主题

**给三个选项，选一个你最感兴趣的：**

**选项A：一个人在做一件事**
- 例如：老人下棋、小孩吃面、女人梳头
- 难度：★★☆（单人场景，适合第一次）
- 叙事重点：动作 + 道具 + 环境暗示

**选项B：两个有关系的人**
- 例如：母女在公园、朋友在咖啡馆聊天、小贩和顾客
- 难度：★★★（需要处理人物关系和互动）
- 叙事重点：视线方向 + 距离 + 肢体语言

**选项C：一个有情绪的场景**
- 例如：雨天公交站、深夜便利店、清晨菜市场
- 难度：★★★★（需要氛围营造）
- 叙事重点：光影 + 环境 + 人物姿态

> **建议：** 第一次选A。把完整流程走通比挑战难度更重要。

## 七天碎片时间创作计划

### Day 1（周一）：选题与收集素材 — 通勤+午休 30分钟

**任务：** 确定画什么，收集参考。

- 在地铁上观察周围的人，在心里"拍"三张照片（用叶浅予的"目识"法）
- 到公司后，花10分钟在网上找1-2张接近你想画场景的照片
- 在速写本上写下你的主题，一句话：
  - ❌ "画一个人"（太空泛）
  - ✅ "午后，老人在棋摊边沉思，手悬在棋盘上方"（有画面感）

**输出：** 一句主题描述 + 1-2张参考图

### Day 2（周二）：构图草稿 — 晚间 20分钟

**任务：** 画3个构图小稿（thumbnail），选一个。

每个小稿不超过5分钟，尺寸约5×7cm。关注的不是细节，是：

1. **主体位置：** 放在正中？偏左？偏右？（建议偏左或偏右，更动态）
2. **视线方向：** 人物朝哪看？视线方向留出空间还是堵住？
3. **环境信息：** 道具画多少？背景画到哪？
4. **留白规划：** 哪里不画？

画完三个，选一个最有感觉的。在旁边标注：
- 画幅最终尺寸（建议16K竖构图或横构图）
- 主要色调（如果是水墨/线面结合）

**输出：** 3个构图小稿 + 1个最终选择

### Day 3（周三）：大形与骨架 — 晚间 25分钟

**任务：** 在正式画幅上定骨架。

用极轻的线条（2H铅笔或浅色笔）画出：

1. **视平线：** 画面中眼睛高度的水平线
2. **主体动态线：** 从头到脚的一条贯通线条，定姿势
3. **体块概括：** 头（椭圆）、胸腔（方盒）、骨盆（方盒）、四肢（圆柱）
4. **道具位置：** 棋盘、椅子、桌子等大致轮廓
5. **背景边界：** 标记背景画到哪里停止

> **关键提醒：** 这一步是骨架。骨架歪了，后面画再精细都救不回来。参考伯里曼体块法和几何形体概括（见 sketch/03 和 sketch/10）。

**输出：** 一张有完整骨架线的草稿

### Day 4（周四）：主体深入 — 晚间 30分钟

**任务：** 从主体人物开始，画出确定的线条。

工作顺序：**头→颈→肩→躯干→手臂→手→腿→脚**

注意几个要点：

- **从上到下画：** 头部先确定，后面比例就有参照
- **果断下笔：** 骨架已经定了位置，这一步是"确认"不是"试探"
- **留有余地：** 手和脸可以稍微细化，但不要钻进去画细节——整体关系比局部精彩更重要
- **边画边调：** 发现骨架的问题现在改还来得及

> **常见陷阱：** 90%的人在这一步会忍不住去画脸的细节。忍住。先把全身的线条通一遍。

**输出：** 主体人物线条完成稿

### Day 5（周五）：环境与道具 — 晚间 25分钟

**任务：** 画入道具、环境暗示、背景元素。

这一步决定你的画是"一个人站着"还是"一个人在一个场景里"：

1. **核心道具：** 画出与人物有互动的道具（棋盘、杯子、书）
2. **环境暗示：** 不需要画完整背景，几条线暗示空间即可
   - 地面线：一条横线交代人物站在哪
   - 家具暗示：椅子的靠背、桌子的边
   - 垂直参照：墙角线、柱子、树干
3. **质感区分：** 用不同线条质感区分材质
   - 皮肤：流畅曲线
   - 衣物：跟随形体的弧线
   - 木头/金属：直线为主
   - 背景：更轻、更虚的线条

> **梁楷减笔思维（见 appreciation/007）：** 背景不需要画完。一两根线交代空间就够了，剩下的让观者用想象补全。

**输出：** 有环境和道具的半成品

### Day 6（周六）：调整与深入 — 周末 40分钟

**任务：** 整体审视，调整关系，局部深入。

这是最关键的一天——**退远看**。

- 把画放在1米外，眯着眼看整体：
  - 主体突出吗？还是被背景抢了？
  - 左右平衡吗？上下比例舒服吗？
  - 视线方向有呼吸空间吗？
  - 最深的暗部和最亮的留白分布合理吗？

发现问题后：
- **太空的地方：** 加一根环境线或道具暗示
- **太满的地方：** 橡皮擦掉一些背景线条
- **不够聚焦的地方：** 加深主体关键位置的线条

然后选择**1-2个焦点区域**深入刻画：
- 人物的脸部表情
- 与道具交互的手
- 衣纹的关键转折

> **注意：** 焦点不要超过2个。处处深入 = 处处平淡。蒋兆和的《流民图》虽然人物众多，但每幅的焦点都非常明确（见 appreciation/006）。

**输出：** 90%完成的作品

### Day 7（周日）：收尾与签名 — 周末 20分钟

**任务：** 最后的调整、签名、拍照存档。

收尾清单：

- [ ] 最后一遍退远看，做微调
- [ ] 检查有没有"半根线"（画了一半没接上的线条）
- [ ] 确认留白区域干净（没有杂乱的浅线）
- [ ] 签名 + 日期（角落，小而克制，不抢画面）
- [ ] 拍照或扫描存档
- [ ] 在背面写一行创作笔记（用了什么技法、哪里满意、哪里想改进）

> **签名建议：** 用比画面更细的笔签。位置选右下角或左下角。大小不超过画面短边的1/20。

**输出：** 一幅完整的速写作品 ✅

## 常见问题

### Q：画到一半觉得画砸了怎么办？

**继续画完。** 这是最重要的建议。

第一幅作品的目标不是"画好"，是"画完"。画到一半发现比例歪了、构图不满意——记下来，下一幅改。这一幅硬着头皮走完。你会发现，很多"画砸了"的地方，在整体完成后反而变成了画面的生动之处。

史国良的藏区写生中，大量作品都有"不准确"的地方，但生命力远超精确的人体结构图（见 appreciation/008）。

### Q：要不要用橡皮？

尽量少用。画错的线不要擦——在旁边重新画一根对的。叠在一起的线条反而有速写的生动感。

如果杂线实在太多影响判断，可以轻轻擦掉一些，但不要追求"干净"。速写的魅力之一就是过程可见。

### Q：画多大合适？

第一次建议16K（约18×26cm）。太小了细节画不进去，太大了容易陷入局部。16K足够你施展，又不会大到有压力。

### Q：用铅笔还是钢笔？

**第一次用铅笔。** 铅笔可以修改、可以渐变、可以轻可以重，容错率最高。等你走完一次完整流程，第二次可以尝试钢笔（不可修改，逼你果断）。

## 完成后的复盘

画完之后，花5分钟做一个简单复盘。在速写本背面写下：

**三个满意的点：**
1. （例如：构图选择偏左，留白效果不错）
2. （例如：手的姿态抓住了"悬在棋盘上方"的犹豫感）
3. （例如：背景只画了一根椅背线，空间感反而出来了）

**一个想改进的点：**
1. （例如：头部画得偏大，比例下次注意）

**下一步：**
- 这幅作品用了哪几种技法？（白描？线面结合？水墨？）
- 下一次想尝试什么不同的？（更大的画幅？两个人？水墨渲染？）

## 延伸思考

### 这幅作品与练习稿的关系

你画的这幅完整作品，其实是之前所有练习的"汇演"：

| 你用到的能力 | 来自哪篇练习 |
|-------------|-------------|
| 构图判断 | sketch/19 画面叙事性 |
| 比例和形体 | sketch/03 几何形体概括 |
| 体块骨架 | sketch/10 体块化训练 |
| 线条质量 | sketch/01 握笔与线条基本功 |
| 留白意识 | sketch/09 负空间观察 |
| 主题记忆 | sketch/11 目识心记意测 |
| 减笔取舍 | appreciation/007 梁楷 |

### 叶浅予体系的创作闭环

这幅七日创作计划，本质上就是叶浅予目识心记意测体系在创作层面的完整应用：

- **Day 1-2 目识：** 观察收集素材，确定画什么
- **Day 3 心记：** 用骨架和体块记住结构关系
- **Day 4-6 意测：** 基于理解进行创作表达
- **Day 7 呈现：** 收尾、签名、交付

### 下一幅怎么画

第一幅完成后，你会对整个流程有体感了。第二幅建议：

- **换一个主题类型：** 如果第一幅选了"单人做事"，第二幅试"两人关系"
- **换一种工具：** 如果第一幅用铅笔，第二幅试钢笔或毛笔
- **缩短周期：** 从七天压缩到四天（周末完成主要部分）
- **提升完成度：** 在焦点区域多花10分钟

> **终极目标：** 当你能在一周内稳定产出一幅完整的速写作品，你就从"学画的人"变成了"在创作的人"。这个身份的转变，比任何技法突破都重要。
