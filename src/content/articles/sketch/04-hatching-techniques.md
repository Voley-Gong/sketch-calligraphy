## 为什么排线是速写的分水岭

前面三章我们解决了：怎么握笔、怎么观察、怎么用几何体概括形。但画出来的东西还是"扁"的——因为你只有轮廓线，没有**明暗体积**。

排线（Hatching）就是用**一组有组织的线条**来表现光影、体积和质感的技法。它是从"线"到"面"的桥梁。

速写的排线和素描的排线不完全一样：

| | 素描排线 | 速写排线 |
|--|---------|---------|
| 目的 | 精确还原光影层次 | 快速暗示体积和光源方向 |
| 线条密度 | 高，多层叠加 | 低，2-3层为主 |
| 线条长度 | 短而均匀 | 长而自由 |
| 时间成本 | 高（一幅画数小时） | 低（一幅画 5-30 分钟） |

速写排线追求的是**"一笔暗示一个面"**——不需要画到位，但需要让观众的眼睛"脑补到位"。

## 一、三种核心排线

### 1. 平行排线（Parallel Hatching）

**最基础、最实用的一种。**

画法：同一方向、大致平行的一组线条，间距均匀。

```
//////////
//////////
//////////
```

**要点：**
- 线条方向应**顺应物体的形体走向**。画一个圆柱体，排线方向就应该是弧形的，而不是直上直下
- 间距越密 → 颜色越深 → 表示暗面
- 间距越疏 → 颜色越浅 → 表示亮面或灰面
- 每根线的起笔和收笔要轻（"两头轻中间重"），避免出现"井"字形的生硬边界

**练习方法：**
1. 在纸上画一个 5cm × 5cm 的方框
2. 用签字笔从左到右画平行斜线，间距约 3mm
3. 要求：线条间距均匀、方向一致、两端轻中间重
4. 每天画 5 组，连续 3 天——直到手感自动化

### 2. 交叉排线（Cross Hatching）

**在平行排线的基础上叠加另一方向的排线，加深暗度。**

```
//////////
//////////
\\\\\\\\\\
\\\\\\\\\\
```

**要点：**
- 第一层和第二层的夹角以 **45°-60°** 为宜，太小会糊在一起，太大则显得生硬
- 每加一层，暗度增加一个"色阶"——通常 3 层足够表现从亮到暗的全范围
- 不要一上来就叠很多层，速写中**两层交叉排线**已经能解决 80% 的明暗问题

**层次关系：**
- 第 0 层（留白）= 高光/亮面
- 第 1 层（平行排线）= 灰面
- 第 2 层（交叉排线）= 暗面
- 第 3 层（再交叉一层）= 最暗部/投影

### 3. 渐变排线（Gradated Hatching）

**从密到疏平滑过渡的排线，表现光影的渐变。**

```
||||
|||||
||||||
|||||||
||||||||
```

（从右到左间距渐密，表示从亮到暗的过渡）

**要点：**
- 渐变的"节奏"要均匀——间距变化是渐进的，不要突然跳变
- 速写中最常用的渐变场景：球体的明暗过渡、圆柱体侧面的明暗过渡、皮肤上的柔和阴影
- 渐变排线是三种里最难掌握的，需要手对"间距"的精细控制

## 二、排线方向与形体走向

这是排线最核心的知识点：**线条不是随便画的，它必须"贴合"物体的表面。**

### 规则：线随形走

想象你用手掌抚过一个物体表面，排线就是你手指划过的痕迹。

| 物体类型 | 排线方向 | 直觉理解 |
|---------|---------|---------|
| 球体 | 弧线，围绕明暗交界线弯曲 | 像地球仪上的经纬线 |
| 圆柱 | 沿侧面纵向弧线 | 像包裹在管子上的线条 |
| 方体 | 直线，沿表面方向 | 方体的面是平的，排线也是直的 |
| 不规则形体（人体等） | 沿肌肉/骨骼走向 | 这是速写里最需要练习的 |

### 最常见的错误

新手最容易犯的错误是：**不管画什么，排线方向都一样**——全部 45° 斜线。

这样画的后果：看起来像给物体贴了一层"百叶窗"，体积感全无。因为排线方向没有配合形体，反而打断了视觉对"曲面"的感知。

正确做法：每画一组排线之前，先问自己——**"这个面的走向是什么方向？"** 然后让排线沿着那个方向走。

## 三、速写中的排线策略

速写不需要像素描那样精细的光影，排线的目标是**"暗示"而非"还原"**。

### 策略一：只画暗面

用一层平行排线（局部加一层交叉）标出暗面和投影，亮面全部留白。这是速写最常用、最高效的明暗策略。

**优点：** 速度快、体积感强、画面干净。
**适用场景：** 阳光下的建筑、有明确光源的人物、静物速写。

### 策略二：沿轮廓线密集排线

在物体轮廓线内侧画密集排线，向内快速渐变到留白。这种手法接近中国传统绘画中的"淡墨渲染"。

**优点：** 既有体积感，又保留了线条的表现力。
**适用场景：** 人像速写、手部/脚部细节。

### 策略三：自由方向短线群

不追求方向统一，用方向各异、长短不一的短线快速堆积出暗部色调。

**优点：** 画面生动、有手绘的温度感。
**适用场景：** 快速速写（1-3 分钟）、户外写生。

## 常见问题

### Q1：排线总是画不均匀，间距忽大忽小怎么办？

**答：** 这是正常的。均匀排线是一种需要"肌肉记忆"的技能，刚开始靠有意识控制，练多了会变成自动化动作。建议：放慢速度，每画一根线都"预判"它的位置和间距，而不是凭感觉快速排。宁可慢而匀，不要快而乱。

### Q2：速写中排线和轮廓线的关系是什么？

**答：** 轮廓线是"骨架"，排线是"血肉"。先用轻线确定轮廓，再用排线填充暗部。注意：**暗面一边的轮廓线应该更重、更明确**，亮面一边的轮廓线应该更轻、甚至消失——这就是"实线"与"虚线"的处理，也叫"线有轻重"。

### Q3：用签字笔/中性笔画排线，没法像铅笔那样擦抹怎么办？

**答：** 签字笔确实不能用擦抹过渡，这恰恰是它的优势——逼你用**线条本身**来控制明暗，而不是靠擦抹偷懒。签字笔排线的核心就是"间距控制明暗"。如果你觉得一组排线不够暗，不是加重力度（签字笔加不了），而是叠第二层交叉排线。

## 练习建议

### 通勤时间（地铁上）

- **观察练习**：看窗外建筑或车内人物，用眼睛"追踪"各表面的明暗交界线位置，想象自己会在哪里画排线
- **手指练习**：用食指在手机屏幕上（息屏状态下）画排线动作，体会间距均匀感

### 午休时间（5-10 分钟）

- **方块明暗练习**：画一个小方块（3cm），确定一个光源方向，用排线画出暗面和投影。一幅 2 分钟，画 5 个不同光源方向（左上光、正上方、右下光等）

### 晚间练习（15-20 分钟）

1. **纯排线训练**（5 分钟）：画 5 个方框，分别练习平行排线、交叉排线（两层）、渐变排线，每组填满一个方框
2. **几何体应用**（10 分钟）：画一个球体和一个圆柱体，用排线标出明暗交界线两侧的明暗变化。球体用弧形排线，圆柱用纵向弧线
3. **简单物件**（5 分钟）：画一个苹果或杯子，用"只画暗面"策略快速上明暗

## 延伸思考

排线的本质是**用线表现面，用面表现体**——这是中国画里早就有的智慧。中国传统绘画的"皴法"（如披麻皴、斧劈皴），其实就是山水画中的"排线技法"：用有组织的线条来暗示山石的体积和质感。

具体来说：
- **披麻皴** ≈ 平行排线，表现柔和的土质山体
- **斧劈皴** ≈ 短而有力的交叉排线，表现坚硬的石质山体
- **雨点皴** ≈ 点状排线，类似点彩暗示明暗

当你理解了排线的原理，再去看中国山水画，会发现古人的"皴法"就是一套经过千年打磨的排线体系。西方速写的排线和中国画皴法在这里是**同一个知识的不同表达**。

掌握了排线，下一章我们将进入 **明暗五大调**——光影逻辑的系统框架，这是让任何物体"立体起来"的核心理论。
