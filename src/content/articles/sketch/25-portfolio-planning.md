## 为什么需要作品集

你画了三个月，速写本翻了五本，硬笔练习纸攒了一叠。有一天朋友问："能看看你的作品吗？"你翻开速写本——第一页是三个月前画歪的杯子，第三页是涂改三次的人体，第七页画到一半就放弃了。

**单张画是你的技能证明，作品集是你的审美名片。**

作品集不是"所有画的合集"，而是你从大量练习中**策展**出来的精华。它体现三件事：你的技术水平、你的审美眼光、你的自我判断力。

门采尔之所以被尊为大师，不只因为他画得多，更因为他知道哪些画值得拿出来。他的展览作品和速写本里的草稿之间，存在严格的筛选。

## 第一步：选作品——少即是多

### 数量原则

| 用途 | 张数 | 说明 |
|------|------|------|
| 社交媒体发图 | 3-5张 | 一组图讲一个主题，不要凑数 |
| 比赛/展览投稿 | 3-8张 | 严格按征稿要求，宁可少不要烂 |
| 个人作品集（PDF） | 12-20张 | 完整展示技能范围和个人风格 |
| 求职/接单作品集 | 8-15张 | 针对客户需求精选 |

**核心原则：一张好画比十张平庸之作有说服力一百倍。** 如果你有20张画，其中3张很好、5张一般、12张平庸——只放3张好的。

### 选作品的四维评估法

每张候选作品从四个维度打分（1-5分），总分16分以上才入选：

| 维度 | 评分标准 | 自问 |
|------|---------|------|
| **技术完成度** | 造型准、线条稳、明暗合理 | "这张有没有明显的硬伤？" |
| **风格辨识度** | 一眼能认出是你画的 | "和别人放一起，能认出来吗？" |
| **情感表达力** | 画面传递了某种情绪或故事 | "看的人会有感觉吗？" |
| **完整性** | 不是半成品、没有明显涂改 | "能挂上墙吗？" |

**操作方法**：把候选作品摊开（或排列在屏幕上），逐张打分。低于14分的坚决淘汰。犹豫的也淘汰——犹豫本身就是答案。

### 作品类型搭配

一本好的作品集不应该只有一种画。建议的搭配比例：

- **主体作品**（60%）：你最擅长方向的最佳发挥
- **探索性作品**（25%）：不同风格/媒介的尝试，展示可能性
- **研究性作品**（15%）：手部研究、结构练习等，展示思考深度

例如一本速写作品集的选片：
- 3张街景速写（主体，水墨风格）
- 2张人物动态速写（主体，线面结合）
- 1张白描花卉（探索，纯线条）
- 1张夸张变形人物（探索，风格化）
- 2张手部/眼睛结构研究（研究性）
- 1张30秒动态速写组图（研究性，展示观察力）

## 第二步：排版——让作品自己说话

### 排版的黄金法则

**画是主角，排版是画框。** 排版的存在感越低，作品越突出。

| 原则 | 说明 | 常见错误 |
|------|------|---------|
| 留白充足 | 每张画周围至少留画面宽度1/3的白 | 把画塞满页面，喘不过气 |
| 统一背景 | 全部白底或全部浅灰底，不要混用 | 有些白底有些灰底，杂乱 |
| 对齐一致 | 画的底边或中线对齐 | 每张画位置随意，像贴照片 |
| 大小有节奏 | 大小画交替排列，不要全部一样大 | 全部一样大，像产品目录 |

### 单页 vs 对页

- **单页单图**：适合精品作品，给足空间，让画面独立呼吸
- **对页双图**：两张有关联的作品并排，产生对话感（如一张铅笔稿+一张完成稿）
- **组图排列**：4-6张同类小画网格排列，展示系列感（如30秒动态速写组图）
- **细节放大**：大图旁边放一个局部放大，展示细节刻画能力

### 叙事顺序

作品集不是画廊，是一本书。读者会从头翻到尾，所以顺序很重要。

**推荐结构（以12页为例）**：

```
P1  封面/标题页
P2  最强开场（你最好的单张作品，先声夺人）
P3-4  主体系列一（如街景速写 3张）
P5  研究性作品（手部研究等）
P6-7  主体系列二（如人物动态 3张）
P8  探索性作品（不同风格的尝试）
P9  组图（30秒动态速写 6-9张排列）
P10  最强收尾（第二好的作品，留下深刻印象）
P11  创作过程（一张作品从草稿到完成的过程展示）
P12  个人简介/联系方式
```

**核心逻辑**：强开场 → 主体展开 → 变化调节 → 强收尾。就像一场演出，不能把最好的节目全放前面。

## 第三步：数字化——从纸到文件

### 扫描/拍摄规范

| 方法 | 适合 | 要求 |
|------|------|------|
| 扫描仪 | 铅笔/钢笔画 | 300dpi以上，灰度模式，白底 |
| 手机拍摄 | 大幅/装裱作品 | 自然光下正面拍摄，避免阴影和反光 |
| 后期处理 | 所有 | 调整白平衡、对比度，忠实于原画 |

**常见问题**：
- 铅笔稿扫描发灰 → 扫描后用色阶工具把白场拉到纯白
- 纸张纹理干扰 → 扫描时用纸张背面垫底，减少透光
- 手机拍摄变形 → 用扫描App（如vFlat、CamScanner）自动校正

### 输出格式

- **投稿/比赛**：按要求输出（通常JPG，每张不超过5MB）
- **PDF作品集**：横版或竖版统一，文件总量控制在20MB以内（方便邮件传输）
- **在线作品集**：上传递压后的WebP或高质量JPG，适配手机浏览
- **打印备用**：保留300dpi的TIFF/PNG源文件

## 第四步：投稿策略——送到对的地方

### 投稿渠道地图

| 渠道 | 适合 | 特点 |
|------|------|------|
| 速写比赛 | 有竞技意愿的练习者 | 有截稿日期和主题限制，有评审反馈 |
| 美术展览 | 有完整作品的创作者 | 需要实物装裱，有展示空间 |
| 社交媒体 | 所有阶段 | 即时反馈，积累关注，建立个人品牌 |
| 线上画廊/社区 | 有一定作品积累 | 专业人群反馈，如同行互评 |
| 个人网站/博客 | 长期经营者 | 完全自主，作品集的终极归宿 |

### 比赛投稿清单

投稿前逐项检查：

- [ ] **截稿日期**：确认时区，至少提前3天提交
- [ ] **主题要求**：作品是否符合主题（偏题直接淘汰）
- [ ] **尺寸限制**：实物投稿的尺寸范围，数字投稿的像素/文件大小
- [ ] **格式要求**：文件命名、分辨率、色彩模式
- [ ] **参赛资格**：年龄、职业、地域限制
- [ ] **版权条款**：投稿后作品的使用权归属
- [ ] **报名费**：是否有费用，是否合理
- [ ] **评审标准**：评审团是谁，偏好什么风格

### 社交媒体发布节奏

不要一次发完所有作品。建立发布节奏：

- **每周1-2次**：每次1-3张图，配一段简短文字说明
- **主题系列**：一个月围绕一个主题发布（如"六月街景速写"）
- **过程分享**：偶尔发草稿、失败品、创作过程，比只发成品更有吸引力
- **互动**：回复评论、参与话题标签、关注同好

黄胄曾说画画要"多画"，但发表要"精选"。这两个不矛盾——大量练习是输入，精选发表是输出。作品集就是精选的输出。

## 常见问题

### Q：作品数量不够怎么办？

先不急着做作品集。设定一个"作品集目标"：画满50张后，从中选10张。质量不足时硬凑作品集，反而暴露短板。

### Q：早期作品和现在差距很大，要不要放？

除非作为"成长对比"特别展示，否则不放。作品集代表你的**当前水平**，不是你的历史。

### Q：被拒稿了怎么办？

投稿被拒是常态。赛事评审有主观偏好，一次被拒不代表作品不好。看入选作品的风格和水平，判断是技术差距还是风格不匹配，针对性改进。

### Q：数字作品和手绘作品能混在一起吗？

可以，但需要排版上做区隔。比如前半部分手绘、后半部分数字，或者用不同的背景色区分。混排时注意视觉统一。

## 练习建议

**本周作业：迷你作品集（5张版）**

1. 从你所有的画里选出5张最好的
2. 按照四维评估法打分，确认每张都在14分以上
3. 用手机扫描或拍摄，做简单后期
4. 排成一页PDF（横向排列，留白充足）
5. 发给你的朋友或发到社交媒体，看反馈

碎片时间方案：
- **通勤地铁上**（40min）：在手机相册里浏览所有作品照，初筛候选
- **午休时间**（10min）：用打分表给候选作品打分，确定最终5张
- **晚间**（20min）：用手机修图App做简单调色，排版导出PDF

## 延伸思考

作品集策划本质上是**自我策展**。策展（Curating）这个词来自拉丁语curare，意思是"照顾、照料"。你不仅是在照顾你的作品，更是在照顾观看者的体验。

叶浅予的"目识心记意测"体系在这里同样适用：
- **目识**：看到自己所有作品的全貌，不回避不足
- **心记**：记住每张画的创作背景和当时的思考
- **意测**：基于以上，判断哪些作品能代表你，以什么顺序呈现

任伯年的作品之所以"雅俗共赏"，因为他知道不同受众需要看到什么。给你的朋友看和给专业评审看，作品集的选片和排序应该不一样。

赵孟頫提倡"书画同源"，你的速写作品集里也可以放几张硬笔书法——展示你的线条审美来自两个领域的互相滋养。这种跨领域呈现，会让作品集更有厚度。

下一步，当你有了稳定的作品产出和更新节奏，可以考虑建立个人网站作为作品的永久驻地。你手上的这个书画学习网站，本身就是一个最好的作品集载体。
