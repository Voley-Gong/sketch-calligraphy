## 核心原理：画建筑不是画砖头，是画空间

建筑速写初学者最容易犯的错误：从左上角开始，一块砖一块砖地画——画到右边发现透视全错了，整张废掉。

建筑速写的底层逻辑只有一个词：**空间关系**。

建筑物的每一个面、每一扇窗、每一根柱子，都服从于一个不可见的骨架——透视系统。你画建筑的过程，不是"复制外观"，而是"构建空间框架，然后在框架上挂细节"。

这就像[人体速写中的体块化训练]：先搭方盒子骨架，再在上面贴肌肉和皮肤。建筑速写也一样，只不过方盒子就是建筑本身。

## 四阶段进阶系统

### 阶段一：一面墙的透视（入门）

**目标：** 理解"面"在空间中的变化规律。

画一面正面朝向你的墙——这是最简单的：所有线条水平/垂直，没有透视变形。

然后走到这面墙的侧面45度角再看——墙面变成了一个梯形。这个梯形就是透视的起点。

**练习方法：**

1. 找一面有窗户的墙（教学楼、居民楼均可）
2. 正面画一次（纯正面，无透视）
3. 左侧30度画一次（墙面右侧开始收窄）
4. 右侧30度画一次（墙面左侧开始收窄）
5. 对比三张画，感受"面的旋转"

**关键概念：** 水平线在透视中会汇聚到"灭点"（Vanishing Point）。正面看没有灭点（线条平行），侧面看灭点在远处——所有水平线都指向那个点。

> **关联知识：** 这就是[透视基础专项]中讲的一点透视和两点透视的实际应用。如果你对透视原理还模糊，先回去复习那篇。

### 阶段二：一个建筑体量（基础）

**目标：** 把单个建筑当作一个方盒子来画。

任何建筑——无论多复杂——都可以先简化为一个或多个方盒子的组合：

| 建筑类型 | 简化策略 |
|---------|---------|
| 独栋民居 | 一个长方体 + 三角形屋顶 |
| 教学楼 | 一个长方体（多层窗户是表面纹理） |
| 教堂 | 一个长方体（中殿）+ 三角形（山墙）+ 圆柱体（塔楼） |
| 中国古建 | 一个长方体（台基）+ 梯形（屋顶）+ 一排柱子 |

**练习方法：**

1. 先画方盒子轮廓（仅用直线，不画任何细节）
2. 检查透视——所有"应该汇聚"的线是否指向同一个灭点
3. 确认透视无误后，在方盒子表面分割门窗
4. 最后加上屋顶、烟囱等附属结构

**检查清单：**
- [ ] 建筑的垂直线是否真的垂直？（透视中垂直线保持垂直，只有水平线汇聚）
- [ ] 窗户的间距是否"近大远小"？
- [ ] 屋顶的斜线斜率是否正确？

> **德加技法融入：** 德加画建筑/室内场景时喜欢裁切构图——你可以故意把建筑画到纸的边缘然后"切掉"一部分，这种不完整感反而让画面更有现场感。

### 阶段三：一条街道（进阶）

**目标：** 处理多个建筑的透视协调和空间层次。

一条街道是多个建筑并排排列，它们共享同一组灭点（因为都在地面上，朝向相同或相反）。

**核心技术：三层法**

这是从[多人场景速写]迁移过来的方法——建筑速写中的"三层"是：

| 层次 | 内容 | 画法 |
|------|------|------|
| **前景** | 路灯、树、行人、停靠的车 | 线条清晰、细节最多、可以有粗细变化 |
| **中景** | 街道两侧的主要建筑 | 透视框架准确、细节适中 |
| **背景** | 远处的建筑轮廓、天空 | 线条简练、甚至只有轮廓 |

**练习方法：**

1. 选一条你熟悉的街道（上班路上、家附近）
2. 先画一条地平线（视平线）——这是最关键的参考线
3. 确定灭点位置（通常在画面中央偏上的位置）
4. 从中景的主要建筑开始画——先搭方盒子，再添细节
5. 添加前景元素（路灯、树、车）——这些元素框定了画面边界
6. 最后画背景轮廓——简练的几条线就够了

**街道构图的三种经典模板：**

- **对称式：** 你站在街道正中央，两侧建筑向远处汇聚——庄重但有压迫感
- **斜角式：** 你站在街道一侧，画面以斜线构图——动感强、空间感好
- **高视角：** 你从天桥或二楼向下看——能看到屋顶和地面的完整图案

### 阶段四：一个城市片区（高级）

**目标：** 用速写记录城市空间的氛围，而非每个建筑的精确细节。

到了这个阶段，你需要放弃"画得像照片"的执念，转而追求"画得像记忆"。

**核心策略：选择性精确**

这是从安格尔和[多人场景速写]中借鉴的原则——不是所有建筑都同等重要。一张好的城市速写，通常只有1-2栋建筑是"精确"的（有结构、有细节），其余的只是轮廓和暗示。

**五级省略法（从[多人场景速写]迁移）：**

| 级别 | 描述 | 应用场景 |
|------|------|---------|
| Level 1 | 完整结构 + 门窗细节 | 画面焦点建筑 |
| Level 2 | 完整轮廓 + 少量窗户 | 焦点旁边的建筑 |
| Level 3 | 简化轮廓 + 屋顶线 | 中景次要建筑 |
| Level 4 | 仅天际线轮廓 | 远景建筑群 |
| Level 5 | 一根线暗示 | 最远处 |

**练习方法：**

1. 找一个能俯瞰城市的地点（山顶、天桥、高楼窗口）
2. 用10分钟画一张全景——只画轮廓线，不画任何细节
3. 选一栋最有特点的建筑，用10分钟细化它
4. 剩余时间在周围添加简化的建筑轮廓

## 建筑速写的五个关键技术

### 1. 视平线是你的锚

每次开始画建筑之前，先在纸上画一条轻轻的水平线——这就是你的视平线（地平线）。

- 视平线之上的东西，你看到它的底面
- 视平线之下的东西，你看到它的顶面
- 所有灭点都在视平线上

> **这根线就是你的透视坐标系原点。** 没有 it，透视全靠猜；有了 it，透视有据可循。

### 2. 先方后圆

建筑上的弧线（拱门、圆窗、穹顶）不要直接画弧——先画它们的外接方形，然后在方形里"切"出弧线。

这是从[几何形体概括]延续至今的原则：复杂形状永远从简单形状开始。

### 3. 用线条密度暗示材质

| 材质 | 线条处理 |
|------|---------|
| 砖墙 | 短水平线 + 垂直短线的规则排列 |
| 混凝土 | 大面积留白 + 少量裂纹线 |
| 玻璃幕墙 | 水平/垂直长线 + 反光的斜线交叉 |
| 木材 | 顺着纹理的方向性排线 |
| 瓦片屋顶 | 人字形短线的重复排列 |

不需要画每一块砖——画3-5块"暗示"一下，观者的大脑会自动补全剩余部分。这就是[白描速写]中"以少胜多"的思路。

### 4. 树和人是尺度参照

建筑画完后总觉得"不够大"或"太大"？在门口画一个正常人高度的小人，在墙边画一棵树——它们的尺寸会自动校正建筑的比例感。

人和树还有构图功能：
- 前景的人可以遮挡不舒服的建筑底部
- 树的枝叶可以柔化生硬的建筑线条
- 走动的人能给静态建筑增加时间感

> **梁楷减笔法的应用：** 前景人物不需要画五官——一个动态剪影就够了。梁楷的减笔告诉我们：少画一些，观者看到的反而更多。

### 5. 天空不是空的

城市速写的天空往往占了画面30-50%的面积——不要让它空白。

- 几朵简笔云（3-4条弧线即可）能立刻增加画面深度
- 一根电线（一条横穿画面的细线）能打破天空的空白
- 远处飞过的鸟群（几个"V"形）能给画面注入生命

> **夜景建筑速写特别提醒：** 建筑在夜间只剩下轮廓和窗户的光——参考[夜景速写专题]的四步法，先铺暗部（建筑剪影），再画窗户的光点。夜景建筑速写的核心不是画建筑，是画窗户。

## 常见问题

### Q1：为什么我画的建筑总是"歪"？

**根因分析：** 90%的"歪"不是手的问题，是眼睛的问题。你画的是"你认为的水平线"而不是"实际的水平线"。

**纠正方法：** 画之前，拿铅笔水平伸出，闭上一只眼，对准建筑的檐口线——记住这条线在你视野中的角度，然后忠实地画下来。这就是[坐标观察法]在建筑速写中的应用。

### Q2：古建筑的飞檐怎么画？

**答：** 飞檐是中国古建速写中最难的部分。诀窍是"先直后曲"：

1. 先画一条从柱顶到屋脊的直线（假设没有翘起）
2. 在这条线的两端做记号
3. 然后用弧线连接——弧线的最高点在端点外侧偏上
4. 飞檐的弧度不是半圆，是一条"先缓后急"的S曲线

多临摹几幅任伯年或徐悲鸿画中的建筑局部，感受古建线条的节奏。

### Q3：画建筑需要尺子吗？

**答：** 练习时不建议用。尺子画出来的线太"死"，缺少手绘的温度。但可以用尺子做**辅助检查**——画完后用尺子量一量那些"应该汇聚到灭点"的线是否真的对齐。

> 作品创作时可以适当使用直尺画长透视线（很多建筑速写大师也会这样做），但轮廓和细节尽量徒手。

## 练习建议

### 工作日（碎片时间）

**通勤路上（地铁/步行）：**
- 地铁站内画一根柱子 + 天花板的透视线（2分钟速记）
- 出站后选一栋路过的建筑，在脑中"简化方盒子"——不画，只用眼看
- 午休时用手机拍3栋有特点的建筑，晚上根据记忆画轮廓

**网约车上（22:15-22:35）：**
- 透过车窗画街景——车在移动，只能抓印象（这其实是绝佳的"选择性精确"训练）
- 不追求准确，追求氛围——几条线就够了

### 周末（整块时间）

**2小时专项训练：**
- 前30分钟：找一条街，画3个不同角度的方盒子透视图（阶段二练习）
- 中间60分钟：选一个完整街景，用三层法画一张（阶段三练习）
- 最后30分钟：找一处高视角，用五级省略法画一张城市远景（阶段四练习）

### 旅行时

旅行是建筑速写的黄金机会——不同城市的建筑风格差异巨大，是扩展视觉词汇库的最好时机。

**旅行速写三件套：**
1. 一本A5速写本（轻便、不显眼）
2. 一支0.5中性笔（不怕丢、不需要削）
3. 一支折叠铅笔（打草稿用）

**旅行建筑速写的优先级：**
- 第一优先：有地标特征的建筑（一眼能认出"这是哪里"）
- 第二优先：有生活气息的街景（晾衣绳、小店铺、路边摊）
- 第三优先：纯建筑美学（光影、肌理、材质对比）

## 延伸思考

### 建筑速写与人体速写的底层共通

这听起来跨度很大，但底层逻辑惊人地相似：

| 建筑速写 | 人体速写 | 共通原则 |
|---------|---------|---------|
| 先画方盒子体量 | 先画体块骨架 | 复杂形从简单形开始 |
| 透视决定空间感 | 动态线决定运动感 | 抽象骨架优先于表面细节 |
| 五级省略法 | 选择性精确 | 不是所有细节都同等重要 |
| 砖瓦是表面纹理 | 衣纹是表面纹理 | 表面纹理服从底层结构 |
| 树和人提供尺度 | 道具提供情境 | 参照物赋予主体意义 |

**一句话：** 画建筑和画人，都是"先搭骨架再穿衣服"。

### 中国画中的建筑：界画传统

中国画有一个专门的画种叫"界画"——用界尺引线画建筑，追求精确的透视和结构。宋代张择端的《清明上河图》就是界画的巅峰。

界画给我们的启示：
- 中国画家早就理解透视——只不过他们用的是"散点透视"（移动视点），西方用的是"焦点透视"（固定视点）
- 散点透视在长卷建筑速写中极其好用——你可以边走边画，把整条街"拼"在一张纸上
- 界画的线条工整、匀净——这和硬笔书法中"横平竖直"的训练完全一致

### 从建筑速写到水墨建筑

如果你已经学过[水墨速写初探]，可以尝试用毛笔+墨画建筑。建筑特别适合水墨表达：

- 淡墨画轮廓（建筑骨架）
- 浓墨画前景（树、人、路灯）
- 干笔擦出质感（砖墙、瓦片）
- 留白处理天空和水面

**中国画的"计白当黑"在建筑速写中尤其有力：** 你不画的那些窗户、不画的那些砖——恰恰是观者最强烈地感受到"那里有东西"的地方。
