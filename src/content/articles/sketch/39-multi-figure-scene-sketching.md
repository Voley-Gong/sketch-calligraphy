## 核心原理：从"画一个人"到"导演一群人"

画单人速写时，你的大脑处理的是：比例、结构、动态、轮廓。

画多人场景速写时，你的大脑需要同时处理：

1. **每个人的个体准确性**（比例、动态、结构——单人速写的基本功）
2. **人与人之间的空间关系**（谁在前、谁在后、谁被遮挡、重叠多少）
3. **人与人之间的视线与互动关系**（谁在看谁、谁在跟谁说话、谁是孤立的）
4. **人与环境的关系**（人物如何嵌入背景、家具、建筑空间中）
5. **整体的构图节奏**（疏密、虚实、主次——不再是画一个人好不好看，而是整幅画面好不好看）

这就是为什么多人场景速写被称为"从技术到导演的跨越"——你不再只是一个记录者，你是一个**画面导演**。

### 为什么Phase 6必须攻克多人场景

回顾叶浅予的创作方法论（目识→心记→意测），多人场景速写正好对应"意测"的高级阶段：

- **目识**（看清结构）：观察多个人物之间的空间关系
- **心记**（记住关系）：记住谁在什么位置、什么姿势、什么互动
- **意测**（基于记忆创造）：根据记忆中的人物关系，重新编排成有叙事性的画面

蒋兆和的《流民图》就是多人场景速写的终极形态——一百多个人物，各有表情、各有姿态，但整体构成一幅完整的叙事画面。

## 多人场景速写的四大技术支柱

### 一、空间层次管理——"三层法"

多人场景的混乱，90%来自空间层次没有理清。使用"三层法"强制建立秩序：

**前层（画面下方1/3）：**
- 离观者最近的人物
- 细节最丰富、调子最深、轮廓最清晰
- 通常画1-2个人物，作为画面的"锚点"
- 练习时重点：精确性——前层画错了，整幅画的空间感就塌了

**中层（画面中间1/3）：**
- 画面的叙事核心区——主要互动发生在这里
- 细节中等、调子中等、轮廓有虚有实
- 通常画3-5个人物，形成"群组"
- 练习时重点：人物之间的重叠和遮挡关系——谁挡住谁、挡住多少

**后层（画面上方1/3）：**
- 远景人物和环境
- 细节最简、调子最浅、轮廓最模糊
- 可以画很多人，但每个只需要几笔概括
- 练习时重点：克制——后层画太多细节会与前层争夺注意力

**三层法的操作流程：**
1. 先画中层（叙事核心区的1-2个关键人物）
2. 再画前层（叠加在中层之上，部分遮挡中层）
3. 最后画后层（作为背景补充）
4. 检查：眯起眼睛看画面——前层最暗、中层中等、后层最浅，三层次分明

### 二、构图骨架——"疏密节奏"

多人场景的构图核心是**疏密关系**——哪里集中（密），哪里留空（疏）。

**三种基本构图骨架：**

| 构图类型 | 特点 | 适用场景 | 经典案例 |
|---|---|---|---|
| 三角形构图 | 底部密集、顶部稀疏，稳定感强 | 聚会、会议、聚餐 | 达·芬奇《最后的晚餐》 |
| 对角线构图 | 从一角到另一角的流动感，动感强 | 市场、街道、游行 | 门采尔的市场速写 |
| 环形构图 | 人物围成弧形或圆形，中心是焦点 | 讨论、围观、表演 | 席勒的部分群像速写 |

**疏密节奏的具体操作：**
1. 先决定画面的"密区"（信息密集区，通常2-3个人物紧挨）和"疏区"（留白区，通常1个人物或纯背景）
2. 密区用精细线条+丰富调子；疏区用简练线条+大量留白
3. 密区和疏区交替排布——就像音乐的"强拍-弱拍"
4. 画面中至少有一个"呼吸口"（纯留白区域），占画面面积的1/6到1/4

**安格尔的椭圆形体叠加法在这里同样适用**：每个主要人物占一个椭圆空间，多个椭圆在画面中嵌套、重叠、错落——形成有机的构图骨架。参见[appreciation/021]安格尔专题。

### 三、人物互动关系——"视线网"

多人场景最容易被忽略、但最关键的技术：**视线关系**。

一个人看一个方向，是一条线。五个人的视线交织在一起，是一张网。这张网决定了画面的叙事性。

**四种基本视线模式：**

**1. 聚焦型——所有人都看同一个点**
```
    A →   ← B
       ↓ ↑
    C → • ← D
       ↑
       E
```
效果：强烈的焦点感。被看的人或物成为画面绝对核心。
适用：演讲、表演、围观。

**2. 对话型——两人对视，其他人分散**
```
    A ←→ B    C (低头看手机)
              D (看窗外)
```
效果：主体明确但不压迫。有日常感。
适用：餐桌、咖啡馆、办公室。

**3. 散落型——每个人看不同方向**
```
    A→    ←B
    ↑C    D↓
```
效果：疏离感、各自为政。现代生活的"附近的远方人"。
适用：地铁车厢、候车室、图书馆。

**4. 引导型——一个人的视线引导观众看向另一个人**
```
    A → B → C
         ↑
       (观众跟随A的视线看到B，再跟随B的视线看到C)
```
效果：画面内部有"阅读路径"，观众的视线被引导走完一条路线。
适用：叙事性强的场景、街头抓拍。

**操作建议：** 画多人场景之前，先用30秒确定视线模式。把每个人的眼球方向和头部朝向用箭头标注在草稿上——如果这些箭头乱成一团，说明你的叙事还没想清楚。

### 四、省略与概括——"画多少才够"

多人场景最大的技术难点不是画得细，而是**知道该省略什么**。

**省略的五级原则：**

| 级别 | 对象 | 画法 | 用线量 |
|---|---|---|---|
| 一级省略 | 前层主要人物 | 精确刻画五官、手部、动态 | 40%的总用线量 |
| 二级省略 | 中层互动人物 | 概括五官、简化手部、保留动态 | 30%的总用线量 |
| 三级省略 | 中层边缘人物 | 只画轮廓+动态线，不画五官 | 15%的总用线量 |
| 四级省略 | 后层人群 | 用几个椭圆/短线条暗示 | 10%的总用线量 |
| 五级省略 | 最远景 | 2-3笔的色块/线条 | 5%的总用线量 |

**记住安格尔的原则（见[appreciation/021]）：不在所有地方花同样的力气。**

这个五级原则就是安格尔"选择性精确"在多人场景中的具体应用——面部的精细度从中心到边缘递减，用线条的密度梯度来引导观众的注意力。

## 具体方法：从2人到10人的递进训练

### 阶段一：双人速写（2人，推荐2-3周）

**训练目标：** 掌握两个人之间的空间关系和互动关系。

**练习内容：**
- 两个人对话（坐姿对坐）：重点画空间距离和视线关系
- 两个人同行（并排走路）：重点画步态差异和身体朝向
- 一个人站着一个人坐着：重点画高低差和体块大小差

**操作步骤：**
1. 先用30秒画两个人的动态线骨架——两条线确定两个人的动态趋势和空间关系
2. 检查比例：站立的人头长是不是比坐着的人多一个？（因为坐姿压缩了腿长）
3. 再用3-5分钟完善形体和互动细节
4. 最后检查视线关系——两个人的眼睛看向哪里？这个关系清楚吗？

**训练量：** 每天2-3张双人速写，持续2周。

### 阶段二：3-5人群组（3-5人，推荐3-4周）

**训练目标：** 掌握"主次关系"和"构图骨架"。

**练习内容：**
- 咖啡馆3-4人桌（有主对话人+听者+旁观者）
- 地铁车厢一组人（坐成一排，姿态各异但朝向一致）
- 家庭场景（一个大人照顾小孩+旁边另一个大人）

**操作步骤：**
1. 先决定谁是"画面主角"（1人）和"配角"（2-4人）
2. 用三角形或对角线构图骨架安排人物位置
3. 画主角用60%的精力（五官精确、动态丰富）；画配角用40%的精力（概括五官、简化动态）
4. 检查疏密节奏：主角周围是否"密"，画面是否有"疏"的留白区

**训练量：** 每天1-2张3-5人群组速写，持续3周。

### 阶段三：6-10人复杂场景（6-10人，推荐4-6周）

**训练目标：** 掌握"三层法"+"视线网"+完整叙事。

**练习内容：**
- 餐厅/食堂全景（多桌多人、有前景桌椅遮挡）
- 公园/广场场景（人物分散在较大空间中）
- 街道场景（行人+建筑+车辆——需要处理人与环境的关系）

**操作步骤：**
1. **构图规划（1分钟）**：在草稿上画出三层空间分区和疏密节奏。标注"密区"和"疏区"的位置。
2. **中层起手（3-4分钟）**：先画中层核心群组（2-3人），确立画面的叙事焦点。
3. **前层叠加（2-3分钟）**：在前层加1-2个人物（可以部分遮挡中层），画得精确细致。
4. **后层补充（1-2分钟）**：在后层用简练线条画3-5个远景人物，只画动态+轮廓。
5. **视线网检查（30秒）**：检查所有人物的视线方向——是否形成了清晰的视线模式？如果视线散乱，微调1-2个人的头部朝向来理清关系。
6. **整体调整（1-2分钟）**：眯眼看画面——前层最暗最清晰、中层中等、后层最浅最模糊。如果层次不分明，加深前层/减淡后层。

**训练量：** 每天1张6-10人复杂场景，持续4-6周。

### 阶段四：默写与创作（进阶）

**训练目标：** 不依赖眼前实景，凭记忆和想象画多人场景。

这就是叶浅予"意测"阶段的多人群组版本——也是蒋兆和《流民图》的创作方法。

**练习内容：**
- 先写生一个3-5人场景（如咖啡馆）
- 然后收起写生稿，凭记忆重新画一遍同一个场景——你记住了谁？忘记了谁？你记住的人是否就是视觉上最重要的人？
- 再画第三版：同样的场景，但加入2个"你想象出来的"人物——让场景更完整、叙事更丰富

**这个练习的价值：** 训练你"记住重要信息、丢弃冗余信息"的能力。多人场景速写最难的不是画，而是**知道什么值得画**。

## 常见问题

### Q：多人场景速写时，总有人会动、会走开，怎么办？

这恰恰是多人场景速写最有价值的地方——它训练你**快速捕捉+记忆补全**的能力。

策略：
- 先快速画所有人物的动态线骨架（每人10秒，5个人50秒）——这样即使有人走了，你已经有动态趋势的记录
- 然后选择最稳定的人深入画（坐着的人比走路的人稳定）
- 走掉的人用叶浅予"心记"的方法补全——你记住了他的大动态和特征，剩下的靠结构理解来补

**蒋兆和画《流民图》时，一百多个人物不可能是写生出来的——他靠的就是"目识→心记→意测"的完整链路。** 参见[appreciation/006]蒋兆和专题。

### Q：多人场景总是画得太挤或太空，怎么控制？

画之前做"人口规划"：
- 画面面积 ÷ 计划画的人数 = 每个人物的平均空间
- 如果每个人物平均能分到画面面积的1/8到1/12，密度合适
- 如果每个人物分到的空间小于1/15，会太挤；大于1/6，会太空

**更实用的判断标准**：画面中至少有一个区域的面积等于一个人物的大小，是完全留白的。这个"呼吸口"会让整幅画看起来不那么挤。

### Q：前层人物画得太大，把中层和后层全挡住了怎么办？

这是初学者最常见的问题——前层画得太大、太完整，中层和后层被"吃掉"了。

解决方案：
1. **先画中层**，再画前层——这样你会在中层的基础上"叠加"前层，天然形成遮挡关系
2. **前层人物只画2/3**——让前层人物的头部或脚部超出画面边缘，不要画完整。这既解决了"前层太大"的问题，又增强了画面的空间纵深（暗示"画面只是空间的一个切片"）
3. **前层用对角线排列**——前层人物不要排在同一水平线上，让一高一低，形成对角线，中层人物从对角线的缝隙中"露出来"

## 练习建议

### 工作日碎片方案

**地铁通勤（40分钟）——双人速写专项：**
- 上车后扫视车厢，选择两个距离较近的人作为对象
- 每人3分钟，先画动态线骨架，再完善形体
- 每趟地铁画3-4组双人速写
- 重点训练：两个人的空间距离、朝向关系、体量对比

**午休（10分钟）——视线网分析：**
- 在食堂或休息区，用2分钟观察一组人（3-5人）的视线关系
- 在纸上用箭头标注每个人的视线方向和头部朝向
- 分析这组人属于哪种视线模式（聚焦/对话/散落/引导）
- 不需要画完整人物，只画头部位置+箭头——这是"导演视角"的专项训练

**晚间（15-20分钟）——3-5人群组速写：**
- 找一张多人场景照片（杂志、新闻照片、电影截图均可）
- 用10-15分钟画一幅3-5人群组速写
- 重点实践三层法和疏密节奏

### 周末深度方案（60-90分钟）

**户外场景速写：**
- 去咖啡馆、公园、商场等多人场所
- 选一个能同时看到5-8人的位置
- 用45-60分钟画一幅完整的多人场景速写
- 流程：1分钟构图规划→5分钟中层起手→10分钟前层深入→5分钟后层补充→3分钟视线网检查→5分钟整体调整→剩余时间修正和深化

### 周末加餐：大师临摹

**推荐临摹顺序（由简到繁）：**

1. **门采尔的市场场景**（参见[appreciation/002]）——2-3人，学习他的信息量管理
2. **珂勒惠支的战争组画**（参见[appreciation/004]）——3-5人，学习用人物群组表达情感
3. **蒋兆和的《流民图》局部**（参见[appreciation/006]）——5-8人，学习中国画的多人空间处理
4. **叶浅予的舞蹈场景**（参见[appreciation/005]）——3-5人，学习用最少的线条画最生动的多人动态

每幅临摹后，用15分钟做"变体练习"——保留构图骨架，把人物替换成你身边的人物（同事、家人、地铁里看到的人），练习"同一骨架，不同人物"的转换能力。

## 延伸思考

### 多人场景速写与之前所有技法的关系

多人场景速写是Phase 6的"集大成者"——它几乎用到了之前学过的所有技法：

| 已学技法 | 在多人场景中的应用 |
|---|---|
| 握笔与线条基本功（[sketch/01]） | 多人场景需要大量不同轻重的线条——前层重线、后层轻线 |
| 坐标观察法（[sketch/02]） | 多人之间的空间位置关系需要精确测量 |
| 几何形体概括（[sketch/03]） | 每个人物先用几何体块定位，再细化 |
| 30秒动态速写（[sketch/08]） | 先给每个人画动态骨架，再深入 |
| 体块化训练（[sketch/10]） | 多人的体块叠加和遮挡关系 |
| 光影概括（[sketch/12]） | 统一光源下多人的明暗层次 |
| 线面结合（[sketch/13]） | 前层用线面结合深入，后层纯线 |
| 伯里曼体块（[sketch/28]） | 每个人物都是体块组合，多人=多个体块系统的空间共存 |
| 路米斯比例（[sketch/29]） | 远近人物的大小比例用"头长"来校准——远处的人头更小 |
| 透视基础（[sketch/31]） | 多人场景的空间深度依赖一点或两点透视 |
| 头像与表情（[sketch/38]） | 前层人物的面部需要表情刻画 |
| 画面叙事性（[sketch/19]） | 多人场景的视线网和互动就是叙事的核心手段 |

**多人场景速写不是一个孤立的技术——它是你之前所有训练的毕业考试。**

### 书画同源：多人场景与中国画"经营位置"

谢赫六法第五条"经营位置"——就是构图——在多人场景速写中体现得最为充分。

中国画的长卷（如《清明上河图》《韩熙载夜宴图》）处理的就是多人场景的极致形式。它们的构图智慧完全可以迁移到速写中：

1. **散点透视**：不执着于一个固定视点，允许画面不同区域有不同的空间逻辑——速写中可以前层用特写视角、后层用鸟瞰视角，只要整体视觉上和谐
2. **虚实相生**：中国画"计白当黑"——多人场景中的留白区域不是"没画完"，而是刻意给观众的视觉休息区
3. **以线立骨**：多人场景的复杂信息量更需要线条来组织——轮廓线区分前后、动态线表达叙事、视线线连接人物

**赵孟頫"书画同源"理论（见[appreciation/001]）在多人场景中有了新的维度：书法的章法布局（字与字的疏密、行与行的节奏）与多人场景的构图布局（人与人的疏密、层与层的节奏）是同一套审美原则在不同载体上的表现。**

### 下一步：从多人场景到叙事创作

多人场景速写是从"练习"走向"创作"的最后一座桥梁。一旦你能在30分钟内画好一幅8-10人的场景速写，你就拥有了创作叙事性画面的全部技术基础。

下一步的方向：

- **命题创作**（结合[sketch/20]命题创作方法论）：给一个主题（如"等车的人""午餐时间"），用多人场景的方式画出来
- **系列创作**（结合[sketch/21]系列作品构思）：围绕一个场景（如"地铁站"）画一系列不同时段、不同人群的多人场景，形成系列
- **文创场景模拟**（结合[practice/005]命题创作模拟）：模拟雍和宫文创店的场景——给一个文案主题，现场用多人场景的方式创作

**多人场景速写的技术上限极高——你用一辈子去追求也不会碰到天花板。但每进步一点，你看世界的眼光就会多一个维度。**

从画好一条线，到画好一个人，到画好一群人——这就是从技术到艺术的完整路径。
