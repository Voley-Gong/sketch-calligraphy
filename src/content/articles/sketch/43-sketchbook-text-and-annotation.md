## 为什么速写本上应该有文字

大多数速写教学只讲怎么画，不讲怎么在画旁边写字。但翻开任何一位大师的速写本——门采尔、德加、黄胄、史国良——你会发现画面旁边写满了字：日期、地点、对象的名字、色彩标注、技术备注、甚至即兴感受。

这些文字不是"多余的说明"，而是速写生态的一部分：

1. **记忆锚点**：三个月后翻速写本，没有文字标注的画只是一张图，有标注的画是一段完整的记忆
2. **画面构成**：巧妙的文字排列本身就是视觉元素，能平衡构图、引导视线、制造节奏
3. **创作素材**：标注的色彩、感受、观察，是日后创作时最珍贵的原始素材
4. **书画同源**：你练硬笔书法的功夫，直接用在速写本标注上——字和画融为一体

赵孟頫"书画同源"的核心不是"画家也应该练字"，而是**画上的字和画本身是一体的**（[appreciation/001 书画同源](../appreciation/001-painting-calligraphy-unity.md)）。你的速写本就是实践这一理念的最佳场所。

## 四种文字标注类型

### 1. 信息标注（基础版）

**写什么：**
- **日期和时间**：2026.06.28 / 17:30
- **地点**：地铁4号线 / 公司楼下 / 家中
- **对象**：穿红外套的女生 / 路边梧桐树 / 儿子的背影
- **工具**：HB铅笔 / 0.5中性笔 / 炭笔

**写在哪：**
画面左下角或右下角，用小字横排。保持一致性——每次都写在同一个位置，形成个人习惯。

**怎么写好看：**
- 用你练硬笔书法的方法写（[calligraphy/06 速度与美感平衡](../calligraphy/06-speed-and-beauty.md)），日常书写不用太正式，但要工整
- 字号统一，约为画面主体高度的1/10
- 推荐用比画线稍细的笔写标注——画面用2B铅笔，标注用HB铅笔

**范例：**
```
2026.06.28  17:30
地铁4号线  西四→灵境胡同
对面  蓝外套男生  约25岁
HB + 2B  双铅笔
```

### 2. 观察笔记（进阶版）

**写什么：**
- **色彩记录**：你看到的颜色，用文字或色号标注——"领口：群青""皮肤：暖黄偏红"
- **结构分析**：对复杂形体的拆解——"肩=斜方肌三角，从锁骨延伸到耳朵方向"
- **动态感受**：捕捉到的运动趋势——"重心在左脚，右膝微屈，脊椎S形"
- **技法备注**：你用了什么方法——"先铺暗部再提亮""负空间法画椅子间隙"

**写在哪：**
画面旁边的空白区域，用箭头或细线连接到画面的对应位置。像给地图加标注一样。

**为什么重要：**
这是你日后创作最依赖的素材库。当你需要画一个"穿蓝色衣服站着的人"时，翻速写本找到标注了色彩和结构的参考，比拍照片有用得多——因为你标注的是**你亲自观察到的理解**，不是照片的机械记录。

**与叶浅予体系的关联：**
叶浅予的"目识→心记→意测"体系（[appreciation/005 叶浅予](../appreciation/005-ye-qianyu-master-study.md)），"目识"环节的最佳工具就是观察笔记。你的文字标注就是"目识"的物质化——把看到的用文字固定下来，为"心记"和"意测"打下基础。

### 3. 即兴感受（日记版）

**写什么：**
- 画这张画时的感受、心情、灵感
- 旁边发生的小故事
- 突然冒出来的创作想法

**写在哪：**
画面下方或对页（速写本翻过来的那一页），可以写得长一些，像日记一样。

**范例：**
```
今天在雍和宫文创店又看到了那位画家，
他在画一只站立的仙鹤，笔触很松，但结构很准。
我注意到他先用淡墨定大形，再用浓墨"点睛"——
和我现在的练习路线完全一致。
打算下周开始尝试水墨速写。
```

**价值：**
这些文字日后翻阅时，比画面本身更能把你带回那个瞬间。速写本的价值不只在于画技进步，更在于记录了你作为创作者的成长轨迹。

### 4. 画面内文字（创作版）

**写什么：**
- 直接在画面中融入文字元素：招牌、路标、书报标题、人物衣服上的文字
- 创作性排版：把诗句、对话、声音效果融入画面

**为什么高阶：**
这已经从"标注"进入"图文融合"的创作领域。门采尔的工业场景速写中，机器上的铭牌、厂房的标号都是画面的一部分。黄胄的速写中，维吾尔族人物衣服上的花纹和文字装饰本身就是画面节奏。

**与中国画的关系：**
中国画题款传统——画面上题诗、钤印——是"画面内文字"的最高形态。赵孟頫的画上题字不仅是标注，更是画面构图的一部分（[appreciation/010 赵孟頫胆巴碑](../appreciation/010-zhao-mengfu-danba-stele.md)）。你在速写本上的文字实验，就是在为将来可能的"画上题字"做准备。

## 文字与画面的构图关系

文字标注不是随便写的——它在速写本页面上的位置直接影响整体视觉效果。

### 四种排版模式

**模式一：左画右字**
画面在左页，文字标注在右页。适合长篇观察笔记和日记式记录。两页打开时形成"图文对照"的阅读节奏。

**模式二：画中嵌字**
文字直接写在画面内的空白区域。适合短标注（日期、地点、色彩备注）。关键是文字不能破坏画面的视觉重心——写在边角或大面积空白处。

**模式三：围绕式标注**
文字像藤蔓一样围绕画面散布，用细线或箭头连接到对应位置。适合教学笔记和结构分析。看起来像达·芬奇的手稿——图文交织、信息密度高。

**模式四：对角线平衡**
画面偏左上，文字标在右下角（或反之）。文字块起到"视觉配重"的作用，让整页构图平衡。这是最推荐的日常用法——既实用又美观。

### 排版的三个原则

1. **字不能比画抢眼**：标注是配角，画面是主角。字号要明显小于画中主体
2. **留呼吸空间**：文字和画面之间留出至少3-5mm的空白，不要紧贴着画
3. **字体统一**：整本速写本用同一种标注字体风格——要么全用楷书，要么全用行书。不要这一页楷书那一页草书

## 实战：四种场景的标注方案

### 场景一：地铁速写（快节奏）

```
[画面] 对面坐着的人，30秒动态速写
[标注·右下角]
06.28  08:15
4号线  西四→灵境胡同
坐着的男生，戴耳机
重心右倾，左手放膝上
→ 注意肩线左高右低
```

时间紧，标注控制在3行以内。用最精简的信息：时间、地点、对象、一个关键观察。

### 场景二：户外写生（中等节奏）

```
[画面] 公园长椅上的老人
[标注·围绕式]
椅背角度→ 30°后仰
拐杖→ 和右腿平行，形成重复线条
衣服→ 藏青色外套，灰色裤子
[标注·右下角]
06.28  16:00
玉渊潭公园  西门
阴天  漫射光  无明显阴影
HB/2B  速写本8K
```

户外写生时间充裕，可以多写一些。色彩标注和光线条件非常重要——日后创作时这些信息比照片更可靠。

### 场景三：夜景速写（特殊光线）

```
[画面] 夜间路边摊
[标注·画面内嵌入]
光源← 路灯（点光源，暖黄）
        橱窗（面光源，冷白）
人物轮廓 ← 逆光剪影
地面反光 ← 湿润，有倒影
[标注·右下角]
06.28  22:00
家门口路边摊
雨后  地面湿润
→ 用橡皮擦出反光
→ 色粉笔叠加暖黄光源
```

夜景的特殊光线条件必须标注——下次想画夜景时有参考。关联[sketch/40 夜景速写](../sketch/40-night-sketching.md)的四种光源分类。

### 场景四：创作稿（慢节奏）

```
[画面] 命题创作草稿
[标注·左侧]
主题：等待
参考：
  - 蒋兆和《流民图》坐姿（appreciation/006）
  - 安格尔坐像的"选择性精确"（appreciation/021）
  - 修拉的明暗氛围（appreciation/023）
色彩计划：
  主色：暖灰（怀旧感）
  点缀色：冷蓝（等待的冷感）
构图：三角构图，人在左下焦点
[标注·右下角]
2026.06.28
第三稿  调整了脊椎角度
→ 下次试水墨版本
```

创作稿的标注等于完整的设计文档。这些记录让每次创作都有据可循，也为系列创作提供连贯性。

## 常见问题

### Q1：字写得不好看怎么办？

你在练硬笔书法（[calligraphy 系列](../calligraphy/01-eight-strokes-fundamentals.md)），速写本标注就是最好的实战机会。不要怕"写不好看"——越写越好看，而且速写本上的字不需要像书法作品那样正式。重点是**清晰可读+风格统一**。

一个小技巧：先用铅笔轻轻写，确认排版没问题后再描一遍。这和你练字时的"双钩填墨"是一个思路。

### Q2：速写本上写字会不会破坏画面？

不会——前提是你遵循"文字是配角"的原则。三个保障措施：
- 用比画面更细的笔写标注（画面用2B，标注用HB或H）
- 文字写在画面之外或画面的大面积空白区
- 字号控制好——标注文字的高度约为画面主体高度的1/10

### Q3：必须每张都写吗？

不是必须，但建议养成习惯。刚开始会觉得多余，写一个月之后就会变成本能——画完一张如果不写点什么，会觉得"少了点什么"。这就是习惯的力量。

## 练习建议

### 第一周：养成标注习惯

**目标**：每一张速写都至少标注日期+地点+对象。

- **通勤日**：在地铁速写旁边写3行信息标注（日期/地点/对象），控制在30秒内写完
- **晚间**：画完练习后，在旁边写1-2行观察笔记（你在画时发现了什么）
- **工具**：画面用2B铅笔，标注用同一支笔的侧面（轻轻写）或一支HB

### 第二周：观察笔记升级

**目标**：在标注基础上，加入色彩记录和结构分析。

- 选3张本周的速写，在旁边补充详细标注：
  - 用箭头指向画面位置，标注色彩（"外套：群青""皮肤：暖黄"）
  - 记录结构理解（"肩线倾斜15°""重心在左脚"）
- 参考[sketch/06 局部解构法](../sketch/06-local-deconstruction-eyes-hands-feet.md)的分析方式

### 第三周：排版意识训练

**目标**：有意识地设计文字与画面的构图关系。

- 每张速写画完后，先看页面整体——"文字应该放在哪里才能平衡构图？"
- 尝试不同的排版模式：左画右字、画中嵌字、围绕式、对角线平衡
- 找到最适合自己的1-2种模式，固定下来

### 第四周：画面内文字实验

**目标**：尝试把文字融入画面本身。

- 画一个有招牌/路标/书报的场景，把上面的文字如实画出
- 在一张满意的速写旁边题一段话——可以是诗句、感受、或创作想法
- 用[calligraphy/11 行书入门](../calligraphy/11-cursive-introduction.md)的连笔技巧写——行书和速写的气质最搭

## 延伸思考

### 大师的速写本标注风格

- **门采尔**：信息密集型，画面旁边写满技术备注——材料、工艺、建筑结构，像工程师的笔记
- **德加**：几乎不标注，但画面上经常有裁切线和构图辅助线——他的"标注"画在画面里
- **黄胄**：大量标注新疆人物的衣服颜色、民族特征、动作名称——为创作积累素材
- **达·芬奇**：终极范本——镜像文字+图文交织，速写本本身就是艺术品

### 与硬笔书法的共生关系

速写本标注是你每天练硬笔书法的"实战演习场"。你在[practice/012 书画双修打卡](../practice/012-dual-practice-daily-checkin.md)中练的字，立刻用在速写本上——**学和用形成闭环**。

反过来，速写本上的标注需求也会反推你的书法练习——发现自己写不好"2026"的数字、写不好"地铁"的左右结构，就回到[calligraphy/03 偏旁部首结体](../calligraphy/03-radicals-structure-rules.md)和[calligraphy/04 间架结构](../calligraphy/04-frame-structure-proportions.md)中查方法。

### 与创作流程的衔接

当你在[sketch/20 命题创作方法论](../sketch/20-commissioned-creation-methodology.md)和[sketch/21 系列作品构思](../sketch/21-series-artwork-planning.md)中做创作时，速写本上积累的标注就是你的"素材银行"：

- 色彩标注 → 创作时的调色参考
- 结构分析 → 构图时的形体依据
- 即兴感受 → 创作主题的灵感来源
- 观察笔记 → 细节刻画的真实感来源

**一句话总结：速写本上画的画是"你的眼睛"，写的字是"你的大脑"——两者合在一起，才是完整的你。**
