## 从"画皮"到"画骨"

速写初学者最常见的问题：**只画你看到的轮廓线，不理解内部的体积结构。**

结果就是——画出来的人像一个纸片人，没有厚度，没有体积感。换个角度就画不了，因为脑子里没有三维结构。

体块化训练就是解决这个问题的。它的核心思想简单粗暴：

> **万物皆可拆成方盒子、圆柱体、球体的组合。**

这不是新概念。伯里曼（George Bridgman）的人体结构教学就是这个思路，路米斯（Andrew Loomis）也用类似的方法。中国古代画论说"骨法用笔"，虽然语境不同，但本质也是强调：**先理解内在结构，再表达外在形态。**

## 一、三个基础体块

### 1. 方盒子（The Box）

方盒子是体块训练的**核心中的核心**。为什么？因为方盒子有明确的面、棱、角——它能定义空间方向。

一个方盒子告诉你：
- 顶面和底面（上下方向）
- 正面和背面（前后方向）
- 左右侧面

当你把人体胸腔、骨盆、头部都理解成方盒子时，你就知道它们各自"朝向哪里"——这是画动态的基础。

**方盒子练习：**
1. 在纸上画不同角度的方盒子（俯视、仰视、侧视、3/4视角），每个画 2×2 厘米
2. 目标：10 个不同角度的方盒子，面与面之间的透视关系正确
3. 检查标准：每条棱的方向是否合理，面是否"歪"

### 2. 圆柱体（The Cylinder）

圆柱体用来表达**四肢**（手臂、腿）和**颈部**。

圆柱体的关键信息：
- 长轴方向（肢体指向哪里）
- 截面形状（通常是椭圆形，告诉你肢体的朝向）
- 粗细变化（肌肉饱满处粗，关节处细）

**圆柱体练习：**
1. 画一根弯曲的手臂：用两个圆柱体连接（上臂 + 前臂），中间用肘关节的圆球连接
2. 注意圆柱体的截面椭圆——它定义了手臂的朝向
3. 画 5 组不同姿态的手臂和腿

### 3. 球体（The Sphere）

球体用来表达**头部**（简化）、**关节**（肩、肘、膝、腕）。

球体的优势：它没有方向性，可以任意旋转。配合一个"中轴线"，就能定义头部的朝向。

**球体 + 中轴线练习：**
1. 画一个圆，加一条十字线（纵轴 + 横轴）
2. 纵轴 = 面部中线（鼻梁所在线）
3. 横轴 = 眉毛线 / 眼睛所在的弧线
4. 旋转中轴线，模拟头部不同朝向：正面、3/4、侧面、仰视、俯视
5. 画 8 个不同朝向的"球体头"

## 二、人体体块化拆解

这是体块化训练的主战场。把人体拆成以下模块：

```
人体体块简表
├── 头部 → 球体（带中轴线）
├── 颈部 → 圆柱体（短粗）
├── 胸廓 → 方盒子（大，上窄下宽）
├── 骨盆 → 方盒子（比胸廓略小）
├── 脊柱连接 → 胸廓与骨盆之间的区域是腰部
├── 上臂 → 圆柱体
├── 前臂 → 圆柱体
├── 手 → 扇形/楔形（简化）
├── 大腿 → 圆柱体（粗）
├── 小腿 → 圆柱体（细）
└── 脚 → 楔形
```

### 关节是"铰链"

把关节理解为铰链点：
- 肩 = 球窝关节（大范围旋转）
- 肘 = 屈伸关节（主要弯曲）
- 腕 = 多方向关节
- 髋 = 球窝关节
- 膝 = 屈伸关节
- 踝 = 多方向关节

画动态时，先定关节位置，再用圆柱体连接——就像搭骨架。

### 体块化画人的步骤

**Step 1：定中线**
画一条从头顶到脚底的动态线（中线）。这条线决定全身的动势。

**Step 2：放体块**
在中线上依次放置：头（球）→ 胸廓（盒子）→ 骨盆（盒子）。注意三个体块的**朝向不必一致**——身体的扭转感就来自胸廓和骨盆的角度差。

**Step 3：连四肢**
用圆柱体连接四肢。先定关节位置（肩、肘、腕、髋、膝、踝），再画圆柱。

**Step 4：加细节**
在体块框架上添加肌肉线条、面部、衣物。这一步才是"画皮"。

## 三、体块化思维迁移到非人体

体块化不止用于画人。万物皆可：

| 对象 | 体块拆解方式 |
|------|-------------|
| 猫/狗 | 头（盒）+ 身体（盒）+ 四肢（柱）+ 尾巴（锥） |
| 汽车 | 车身（扁长盒）+ 车顶（梯形盒）+ 车轮（圆柱） |
| 树 | 树干（锥/柱）+ 树冠（球/多个球组合） |
| 建筑 | 基本就是方盒子的组合 |
| 手 | 手掌（盒）+ 手指（短柱 × 5） |

**核心心法：先问"这是什么形状的组合"，再动笔画。**

## 四、透视基础：让体块"站得住"

体块化画法的成败取决于透视。你不需要学复杂的三点透视，但必须掌握一点：

### 一点透视速记

所有平行于视线的线汇聚到一个**灭点**。

```
     ┌─────┐
    /│     │\
   / │     │ \
  /  │     │  \
 /   │     │   \
     ↓灭点
```

### 实用规则（画体块时）

1. **正面平行于画面的盒子**：只有前后方向的棱汇聚到灭点，其他棱保持水平/垂直
2. **角度越大，面越窄**：盒子的一个面转向侧面时，正面会变窄，侧面会变宽
3. **越远越小**：这是废话，但画的时候经常忘

**练习：** 画一排等大的方盒子，从左到右逐渐远去（角度可以微变）。检查灭点是否一致。

## 常见问题

### Q：体块化画出来的人很僵硬怎么办？

这是必经阶段。先用体块搭框架（接受僵硬），再在框架上加动态线和肌肉线条。**先正确，再生动。** 30秒动态速写（上一篇）可以作为体块训练的"解药"——一严谨一放松，交替练习。

### Q：我画几何体没问题，但换成人体就不知道怎么套怎么办？

这说明你对人体各部分的**位置关系**还不够熟悉。解决方案：
1. 找一张人体照片，在上面用红笔叠画几何体
2. 画 20 张不同姿态的照片叠画
3. 然后去掉照片，纯靠体块画相同姿态

这个"描红→临摹→默写"的路径最有效。

### Q：伯里曼和路米斯的方法有什么区别？

- **伯里曼**：更强调体块的**楔合关系**——体块和体块之间怎么扣在一起。风格更"粗犷有力"。
- **路米斯**：更强调**比例和测量**——用头长作为单位，全身 = 7.5 或 8 个头长。风格更"经典学院"。

建议：先学路米斯打比例框架，再学伯里曼理解体块穿插。两者互补。

## 练习建议

### 通勤时（地铁/网约车上）

**手指体操版体块训练：**
用手指在腿上"画"几何体——想象方盒子的12条棱，逐条"描画"。重点感受棱的方向变化。不引人注目，但能有效建立空间感。

### 午休时（10分钟）

**照片叠画：**
手机存 3-4 张运动员/舞者的照片。午休时用手机自带的标注功能（或备忘录画笔），在照片上叠画几何体——标出头部球体、胸廓盒子、骨盆盒子、四肢圆柱。

### 晚间练习（20分钟）

1. **5分钟**：画 10 个不同角度的方盒子（纯几何练习）
2. **5分钟**：画 5 个不同角度的球体头部（带中轴线）
3. **10分钟**：用体块法画 2 个人物动态（参考照片或想象）

### 周末加餐

找一段体育比赛视频（足球、篮球、体操），暂停在不同动态画面，用体块法快速画出 5 个姿态。每个 2-3 分钟，不求细节，只求体块关系正确。

## 延伸思考

### 骨法用笔与体块化

谢赫六法第一条"气韵生动"，第二条"骨法用笔"。"骨法"说的是通过笔触传达对象的**内在结构**——这不就是体块化训练的终极目标吗？

中国画虽然不画方盒子，但好的线描每一条线都在暗示内部的体积结构。你体块训练做得越好，将来用纯线条表达时就越有"骨力"。

### 体块化是"世界观"

体块化不只是一种画法，它是一种**看世界的方式**。训练到一定程度，你看任何东西——一个人、一把椅子、一棵树——都会本能地拆解成几何体。这种"X光眼"是所有优秀速写者的共同特征。

而这一切的起点，就是今天画下的第一个方盒子。
