## 核心原理：创作不是灵感，是决策

你看着雍和宫文创店那位画家——给他一个"福"字，他十分钟后交出一幅有人物、有构图、有情绪的画。你以为他靠的是天赋和灵感。

不是。他靠的是**一条跑了上千遍的决策链**。

命题创作的本质：每一个环节都在做选择。选A还是选B，选深入还是选留白，选这根线还是那根线。新手之所以卡住，不是因为手上功夫不够，是因为**不知道该选什么**。

### 决策链 vs 灵感论

| 灵感论（错误心智模型） | 决策链（实际创作过程） |
|----------------------|----------------------|
| 等灵感来了再画 | 拿到主题就开始拆解 |
| 好作品来自天赋 | 好作品来自正确的选择 |
| 每次创作都是全新的 | 每次创作都跑同一条流程 |
| 卡住了 = 没灵感 | 卡住了 = 某个决策节点没想清楚 |

> 这跟编程里的"算法 vs 随机尝试"是一样的。高手不是随机碰对了，是他脑子里的"创作算法"跑得又快又准。

## 五步决策链：拿到主题到锁定方案

### 第一步：主题解码（3分钟）

拿到一个命题，第一步不是画，是**翻译**。

命题通常是一个词、一句话、一个场景描述。你需要把它翻译成**可画的视觉元素**。

**示例：命题"秋日"**

第一层翻译（太抽象，画不了）：
> 秋天、落叶、金黄色的感觉

第二层翻译（有画面了，但模糊）：
> 一个人在公园里走，地上有落叶

第三层翻译（可执行）：
> 一个穿薄外套的中年人，双手插兜，低头走在林荫道上。地上一片片落叶，脚正踩在一片上。光线从左侧斜照，拉出长影。构图竖幅，人物偏右，左侧留给光影和树干。

**解码模板：**

```
命题 → 关键词提取 → 场景具象化 → 人物设定 → 环境元素 → 光线方向 → 构图方向
```

关键原则：**每一步都从抽象走向具体。到第三层翻译时，你应该能在脑子里"看到"这幅画。**

### 第二步：构图选择（5分钟）

主题解码后，你要选择**怎么看这个画面**。

这一步的本质是四个变量：

**变量1：景别**
| 景别 | 效果 | 适用 |
|------|------|------|
| 全景 | 人在环境中，氛围强 | 场景命题（"菜市场""深夜街道"） |
| 中景 | 人物占画面主体，环境是暗示 | 关系命题（"母女""朋友"） |
| 近景/特写 | 聚焦表情或动作细节 | 情绪命题（"沉思""惊喜"） |

**变量2：视角**
- 平视：客观、冷静、纪实感（门采尔常用）
- 仰视：崇高、力量、压迫感（蒋兆和《流民图》常用）
- 俯视：渺小、孤独、全局观（珂勒惠支常用）

**变量3：构图骨架**

四种基本骨架，覆盖90%的速写创作：

```
三角形构图          S形构图
    △                 ~~~
   / \               /   \
  /   \             ~     ~
 /_____\           /       \
主体在下角     主体沿S线分布
稳定、经典       流动、有韵律

对角线构图          框式构图
  ↗                 ┌─────┐
 ↗                  │     │
↗                   │ 主体│
动态、有张力         聚焦、有空间感
```

**变量4：留白方向**
- 人物面朝的方向留白 → 呼吸感、向前看的暗示
- 人物背后留白 → 孤独感、过去式的暗示
- 上方留白 → 崇高、空灵
- 下方留白 → 踏实、接地

> **速判法：** 30秒内选定四个变量。不要反复纠结——选了就画。叶浅予说过："先落笔，错了再改，总比空着纸强。"

### 第三步：线条策略（2分钟）

构图定了，你要决定**用什么线条语言画**。

这一步很多人忽略，但它决定作品的气质。

| 线条策略 | 气质 | 适用主题 | 你的参考 |
|---------|------|---------|---------|
| 纯白描 | 清雅、克制 | 传统、诗意 | sketch/15 白描速写 |
| 线面结合 | 丰富、扎实 | 写实、叙事 | sketch/13 线面结合 |
| 水墨速写 | 意蕴、流动 | 情绪、氛围 | sketch/14 水墨速写 |
| 减笔描 | 极简、有力 | 禅意、瞬间 | appreciation/007 梁楷 |
| 粗线夸张 | 个性、冲击 | 表现、人物 | sketch/16 夸张变形 |

**选择原则：** 你的第一反应通常是对的。看到"秋日"想到白描？那就白描。不要纠结——线条策略是手段，不是目的。

### 第四步：执行节奏（全程的60-70%）

方案锁定后，进入实际作画。命题创作的时间分配和练习稿完全不同：

```
骨架定位 ──→ 主体确立 ──→ 环境补充 ──→ 焦点深入 ──→ 收尾调整
  15%          30%           20%          25%          10%
```

**执行中遵循三个铁律：**

**铁律1：从整体到局部，永远如此。**
先画完所有区域的骨架，再回到头部细化。不要在脸上花了20分钟、身体还空着。

**铁律2：焦点不超过2个。**
一幅画最多有2个"看点区域"。其他地方要学会放手。梁楷的减笔人物画之所以震撼，不是因为画得多，是因为**该放的地方放得彻底**（见 appreciation/007）。

**铁律3：每15分钟退远看一次。**
凑近画太久会丧失整体判断力。退后1米，眯眼看——如果整体关系不舒服，立刻调整，不要"画完再说"。

### 第五步：收尾判断（最后10%）

画到最后10%是最考验判断力的。知道什么时候停笔，比知道怎么画更难。

**收尾检查清单：**

- [ ] 整体退远看——第一眼聚焦在哪？是你想要的位置吗？
- [ ] 有没有"抢戏"的局部？（某根线太重、某块暗部太大）
- [ ] 留白区域够"干净"吗？
- [ ] 人物的视线方向有出口吗？
- [ ] 如果只看画面下半部分——能感受到你想表达的情绪吗？

> **停止法则：** 当你觉得"好像还能再改改"但又说不清改什么——停笔。这个状态下继续画，80%的概率会画过头。

## 常见问题

### Q：拿到命题大脑空白怎么办？

大脑空白 = 你跳过了"主题解码"，直接想画面。

回到第一步：拿出纸笔，把命题写成关键词列表，然后逐个具象化。"秋天"→落叶→人踩落叶→公园→午后光线——画面自然就浮现了。

蒋兆和的创作流程也是从文字构思开始的：先想清楚"要表现什么"，再考虑"怎么表现"（见 appreciation/006）。

### Q：构图总是选不出来，反复改怎么办？

设定一个硬性规则：**只画3个构图小稿，第三个必须定。**

5分钟一个小稿，三个小稿15分钟。第三个画完，掷骰子也得选一个。然后在选定的方案上继续——不准回头改构图。

命题创作的时间约束逼你做决策。犹豫不决不是"追求完美"，是缺乏决策训练。

### Q：画到一半发现构图选错了？

分两种情况：

**情况A：小问题（比例偏了、某个元素位置不好）**
→ 在当前画面上调整。不要推翻重来。

**情况B：大问题（景别选错了、完全没叙事性）**
→ 推翻，但保留第一版的教训。用5分钟画一个新构图小稿，然后快速进入执行。

史国良在藏区写生时，经常遇到画到一半发现构图不对的情况。他的做法是：快速评估，能救就救，不能救就果断重来——但每次重来都比上一版快（见 appreciation/008）。

### Q：命题创作的"完成度"到底要到什么程度？

判断标准：**一个不懂画的人看到这幅画，会觉得"这是完成的作品"还是"这是草稿"？**

具体来说：
- 每个区域都有明确处理（深入过/刻意留白），没有"还没画"的地方
- 主体清晰可辨（不是未完成的人形轮廓）
- 有签名（这让它"正式"成为作品）

不需要：每个细节都精致、背景画满、调子画全。

## 练习建议

### 初级训练：结构化命题（每周1次）

给自己出一个简单命题，用本文的五步决策链走一遍。时间控制在60分钟内。

**命题池：**
- 一杯热茶
- 等车的人
- 雨天
- 午餐时间
- 老人的手

每次画完后，对照决策链做复盘：哪一步犹豫最久？哪一步的选择最终效果最好？

### 进阶训练：限时命题（模拟文创店场景）

给自己一个命题和严格的时间限制：

| 难度 | 时间 | 命题复杂度 |
|------|------|-----------|
| ★★ | 30分钟 | 单人 + 一个道具 |
| ★★★ | 20分钟 | 单人 + 环境暗示 |
| ★★★★ | 15分钟 | 两人关系 + 一个道具 |

> 对应 practice/005 的文创店模拟训练，但这篇增加了"决策链"方法论指导。

### 碎片时间利用

- **通勤路上（40分钟）：** 练"主题解码"。看到什么（广告语、站名、路人），在心里把它翻译成第三层画面描述。
- **午休10分钟：** 练"构图选择"。拿到一个词，30秒内决定景别、视角、骨架、留白方向。
- **晚间15分钟：** 做完整的5分钟构图小稿 × 3，选一个深入到60%完成度。

## 延伸思考

### 命题创作与编程的同构映射

如果你把命题创作和编程做一个对比，会发现惊人的相似：

| 命题创作 | 编程 |
|---------|------|
| 主题解码：把抽象命题翻译成视觉元素 | 需求分析：把用户故事翻译成技术方案 |
| 构图选择：选定景别、视角、骨架 | 架构设计：选定技术栈、设计模式、系统结构 |
| 线条策略：选择白描/线面/水墨 | 代码风格：选择函数式/OOP/响应式 |
| 执行节奏：整体→局部→焦点 | 开发节奏：框架→模块→优化 |
| 收尾判断：知道何时停笔 | 发布判断：知道何时上线（不过度工程化） |

你的客户端技术主程序经验，其实已经训练了你"在约束条件下做最优决策"的能力。命题创作只是把这个能力迁移到视觉领域。

### 与叶浅予体系的完整对接

五步决策链每一步都对应叶浅予的创作理念：

- **主题解码 = 意测的前置条件：** 你必须先想清楚要画什么，才能调动记忆中的素材
- **构图选择 = 心记的空间化：** 把脑子里的画面落实到画面的空间关系
- **执行节奏 = 目识的精细化：** 在观察和记忆的基础上进行视觉表达
- **收尾判断 = 意测的最终检验：** 整体效果是否符合最初的创作意图

叶浅予的完整创作流程（构思→构图→执行→收尾），在这篇决策链中得到了一一对应（见 sketch/11 目识心记意测法）。

### 与大师创作流程的横向对比

| 大师 | 主题解码 | 构图选择 | 线条策略 | 特色 |
|------|---------|---------|---------|------|
| 门采尔 | 工厂/劳动场景 | 平视全景、信息密集 | 线面结合、密集排线 | 细节的史诗感 |
| 席勒 | 情绪/身体 | 近景特写、大量留白 | 纯线条、神经质的抖动 | 情绪的刺痛感 |
| 梁楷 | 禅意/人物 | 大留白、偏心构图 | 减笔描 | 以少胜多的极致 |
| 蒋兆和 | 苦难/生存 | 仰视、纪念碑式 | 线条骨架+水墨血肉 | 沉甸甸的重量感 |
| 史国良 | 藏区/生活 | 中景、叙事性强 | 速写线条+淡彩 | 生活的温度感 |

他们的共同点：**每个人都有清晰的决策模式。** 你现在要做的，是通过反复练习，建立自己的决策模式。

> **终极目标：** 当五步决策链变成肌肉记忆，你拿到任何命题都能在10分钟内进入执行状态——这就是文创店画家的真正秘密。
